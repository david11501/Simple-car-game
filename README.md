# Simple Car Simulation Game

Un joc simplu și interactiv în consolă, scris în Python, care simulează pornirea, oprirea și gestionarea stării unei mașini prin comenzi text.

Acest proiect este un exemplu practic de Finite State Machine (FSM): comportamentul aplicației depinde de starea curentă (mașina pornită sau oprită).

---

## 🎮 Cum se joacă

1. Clonează repo și intră în director:

```bash
git clone https://github.com/david11501/Simple-car-game.git
cd Simple-car-game
```

2. Rulează scriptul (Python 3):

```bash
python simple_car_game.py
```

3. Comenzi disponibile în joc:

- `start`  — pornește mașina
- `stop`   — oprește mașina
- `status` — afișează starea curentă
- `help`   — afișează lista de comenzi
- `quit`   — închide jocul

4. Exemplu de sesiune:

```
Simple Car Simulation. Tastează 'help' pentru comenzi.
> start
Mașina pornește...
> status
Stare: pornită
> stop
Mașina se oprește...
> quit
Sfârșit joc — la revedere!
```

## Cerințe

- Python 3 (nu sunt dependențe externe)
