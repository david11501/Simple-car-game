# Simple Car Simulation Game

A lightweight, interactive console game written in Python that simulates starting, stopping, and managing the state of a car through user commands.

This project is a practical example of a **Finite State Machine (FSM)**, where the application behaves differently based on its current state (whether the car is already running or stopped).
